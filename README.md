# Aatif Khan Pathan — Portfolio Website

> **Cyber-Minimalist dark-mode portfolio** built with Python (Flask) + pure CSS + Vanilla JS.
> Features a live particle / star-field neural-network background, scroll-reveal animations, project filtering, and a contact API.

---

## ✨ Features

| Feature | Details |
|---|---|
| **Particle Engine** | Canvas-based star-field + neural-network node-link animation; mouse-reactive |
| **Scroll Animations** | Staggered reveal on every section as you scroll |
| **Project Filter** | One-click category filtering with smooth transitions |
| **Contact API** | Flask POST endpoint — plug in SMTP / SendGrid to wire real emails |
| **Stat Count-Up** | Numbers animate when the stats grid enters the viewport |
| **Cursor Glow** | Soft radial light that follows the cursor |
| **Fully Responsive** | Mobile-first, hamburger nav, fluid typography |
| **Python-only backend** | Zero Node.js; pure Flask serves HTML generated by Python |

## 🚀 Quick Start

### 1 — Clone / download

```bash
git clone <your-repo-url>
cd portfolio
```

### 2 — Create a virtual environment (recommended)

```bash
python3 -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate
```

### 3 — Install dependencies

```bash
pip install -r requirements.txt
```

Flask is the **only** dependency. No Node, no build step.

### 4 — Run the development server

```bash
python app.py
```

Then open **http://localhost:8000** in your browser.

## 🔧 Customisation

### Update content
Open `data/portfolio_data.py`. Every section of the site is driven by plain Python dicts/lists:

```python
PERSONAL   # name, email, social links, status
STATS      # the six metric cards in the hero
TECH_STACK # categorised skills
EXPERIENCE # timeline entries
PROJECTS   # project cards (set featured=True for violet glow)
```

### Add your resume PDF
Drop your resume into `static/assets/` and update `PERSONAL["resume_url"]` in `portfolio_data.py`.

### Wire up email for the contact form
In `app.py`, find the `contact()` route and uncomment the `smtplib` block. Set environment variables:

```bash
export SMTP_USER="you@gmail.com"
export SMTP_PASS="your-app-password"
```

Gmail users: generate an [App Password](https://support.google.com/accounts/answer/185833) — don't use your main password.

### Change the particle colour
In `static/js/main.js`, edit the `config` object inside `ParticleEngine`:

```js
starColor:  '0, 229, 255',   // cyan stars
nodeColor:  '124, 58, 237',  // violet network nodes
connectDist: 110,            // max connection distance (px)
count:       120,            // total particle count
```

## 🌐 Production Deployment

### Gunicorn (recommended)

```bash
pip install gunicorn
gunicorn app:app -w 4 -b 0.0.0.0:8000
```

### Docker

```dockerfile
FROM python:3.12-slim
WORKDIR /app
COPY . .
RUN pip install -r requirements.txt
EXPOSE 8000
CMD ["python", "app.py"]
```

```bash
docker build -t portfolio .
docker run -p 8000:8000 portfolio
```

### Environment variables

| Variable | Default | Purpose |
|---|---|---|
| `PORT` | `8000` | Server port |
| `DEBUG` | `true` | Flask debug mode |
| `SECRET_KEY` | `change-me-in-production` | Flask secret key |
| `SMTP_USER` | — | Sender email address |
| `SMTP_PASS` | — | SMTP app password |
